Git 是目前世界上最先进的分布式版本控制系统。

## 核心概念

- **分支**: 独立的开发线
- **合并**: 将分支历史整合在一起
- **提交**: 保存项目状态的快照
- **远程仓库**: 托管在服务器上的仓库
- **Pull Request**: 代码审查和合并机制

## 使用经验

日常工作流：
- 功能分支开发
- Git Flow 工作流
- 代码审查
- 冲突解决
