Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时。

## 核心特性

- **事件驱动**: 非阻塞 I/O 模型
- **npm 生态系统**: 丰富的包管理器
- **全栈开发**: 前后端统一语言
- **高性能**: V8 引擎优化
- **跨平台**: 支持 Windows、Linux、macOS

## 使用经验

使用 Node.js 构建：
- RESTful API
- GraphQL 服务
- 实时通信应用
- CLI 工具
- 自动化脚本
